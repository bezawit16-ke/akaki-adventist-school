# 🎓 Student Registration System

A web-based application designed to **register students, collect their necessary documents, and automatically generate ID cards**.  
This project streamlines the enrollment process for schools by combining secure data handling with a simple, user-friendly interface.

---

## 📂 Project Structure

---

## ✨ Features
- **Student Registration**: Collects personal details and required documents.  
- **Document Uploads**: Stores files securely in the `uploads/` directory.  
- **ID Card Generation**: Automatically produces student ID cards with photo and details.  
- **Authentication**: Login/logout system for administrators.  
- **Data Management**: View, edit, and delete student records.  
- **Responsive Design**: Works across devices with clean UI.
